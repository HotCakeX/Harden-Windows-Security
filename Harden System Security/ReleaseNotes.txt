* Added 2 new background customization options to the Settings page: "Custom Mica" and "Custom Blur". They offer you greater flexibility and freedom to customize the app's appearance. You can set a custom background image, apply blur effects on it and more. Animated GIFs also work so you can set a video/animated background for the app.

* Updated dependencies to the latest version.

* Improved the Group Policies Management page: search query will persist after retrieving data. Selected files on the Sidebar of the page can be viewed with right-click or tap + hold.

* Added a new option to the Group Policies Management page to export the policies you see in the list view to JSON. It has CTRL + E keyboard shortcuts too.

* Improved the way partition sizes are displayed in the Bootable Drive Maker page.

* Added a new Security Measure to the Miscellaneous category which will set the Execution Policy of the Windows PowerShell to the most secure state which is "Restricted", completing the following feature request: https://github.com/HotCakeX/Harden-Windows-Security/issues/1140

* When you right-click on the top/title bar of the app, you no longer see the legacy options, instead you'll see a modern flyout menu with Mica style. They are also fully localized so you will see the options in the language you select in the Harden System Security's settings page.
