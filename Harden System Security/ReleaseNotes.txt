* Improved the System Mitigations editing experience. Selecting Modify now automatically retrieves the latest system mitigation settings before opening the editor, ensuring that the displayed configuration is current. Manual refresh remains available through the Refresh button.

* The active badge on the taskbar icon is now displayed for long-running Winget operations.

* Updated dependencies to the latest version.

* You can now select multiple certificates in the Certificate Checking page to delete them from your system. Previously you had to do this one by one. The confirmation message is still displayed for every certificate before deletion.

* You can now filter non-Microsoft-rooted certificates in the Certificate checking page based on chain build success status. This makes the result completely match the SigCheck tool's results from SysInternals, offering a more familiar results while also having options to expand the scope of the scan.

* Improved the app and UI performance.

* You can now choose where the Windows Sandboxes that you create in the Sandbox Maker page are saved to using the newly added UI controls. Completes the following feature request: https://github.com/HotCakeX/Harden-Windows-Security/discussions/1184
