* You can now specify any custom PowerShell commands to run either before or after the custom program launch in the Windows Sandbox maker page. This enables you to create more complex and customized Sandboxes by running scripts that set up the environments exactly as needed before you start working in them.

* Improved the "Manage Installed Apps" page's retrieval performance and the way user information is displayed for each app.

* The Harden System Security app now detects the system languages and if it supports any of them, it will configure its language accordingly. This only happens once at startup when the app is launched on a system for the first time. If user explicitly sets a different language in the app's settings then it will be respected.

* Improved localizations.
